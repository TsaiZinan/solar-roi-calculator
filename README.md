# 光伏收益计算 (PV Profit Calculation)

本项目用于计算光伏、储能与充电桩一体化系统的每日收益及全年预测，通过解析真实电表数据并结合分时电价（峰平谷）策略，生成多场景下的净利润与储能额外增益分析报告。

## 目录结构
- `Script/`: 包含收益计算、年度预测及日度分析的各类 Python 脚本。
- `数据/`: 存放各项目节点输出的 CSV 数据。
- `报告/`: 自动生成的每日收益分析 Markdown 报告以及总收益分析报表。

## 核心功能
1. **每日收益分析 (`daily_analysis.py`)**: 
   - 读取每日 5 分钟间隔的瞬时电表数据。
   - 分析“峰、平、谷”时段光伏产电、储能充放电、负载及电网吞吐情况。
   - 对比计算不同光伏上网电价（0.1元, 0.2元, 0.35元）场景下的总净利润。
   - 隔离计算储能系统的额外收益（通过峰谷套利及减少弃光）。
2. **全年收益预测 (`annual_prediction.py`)**: 
   - 结合天气数据与全年日照情况，预测全年12个月的光储充一体化收益。
3. **利润计算与汇总 (`calculate_profit.py`, `init_summary.py`)**: 
   - 对数据进行深加工，生成详尽的总收益分析报表，方便复盘与管理。

## 每日操作指南

每天进行收益分析前，请按照以下步骤操作：

1. **创建日期目录**：
   在 `数据/` 目录下创建一个以当日日期命名的文件夹，格式为 `YYYYMMDD`（例如：`20260421`）。

2. **放置数据与归档图片（必须）**：
   - **CSV 数据文件**：将**阳光电源储能后台（阳光云）**导出的 5 分钟间隔数据报表（CSV 格式）放入该日期目录中。
   - **归档图片**：将当天**华为智能光伏APP 的每日发电曲线折线图截图**保存为 `YYYYMMDD.jpg` 并放入该日期目录中，用于后续追溯与复盘。

3. **调用 AI 技能执行分析**：
   - 用户**无需手动修改或运行分析脚本**。
   - 只需在对话框中输入 `每日分析` 触发对应 SKILL，AI 助手将自动完成文件的检查与归档、数据合并、利润测算，并自动生成当日分析报告及更新总收益报表。

## 常见问题 (QA)

### Q: 为什么报告中计算的单日充放电量与储能软件中显示的数据（如软件显示充电 437 度 / 放电 404 度）存在微小差异？

**A:** 这个差异（通常在 3% 左右）主要由以下几个维度的原因造成：

1. **离散采样导致的数据积分误差（主要原因）**：
   报告脚本是通过读取 CSV 文件中**每 5 分钟**的一个**瞬时功率快照**进行积分推算（即黎曼求和），而储能软件是由底层智能电表以毫秒或秒级频率连续积分得到的**真实累计电量**。用离散快照推算连续波动的数据必然会产生合理误差。
   
2. **测量口径与效率损耗（交直流差异）**：
   CSV 数据通常测量的是 PCS 交流侧并网点的功率（`储能有功功率(kW)`），而某些储能软件可能统计的是电池端（直流侧）的吞吐量。交流和直流之间的转换存在效率损耗（通常 PCS 效率在 95%-98% 之间），因此导致交流侧数字与内部 BMS 数字有微小偏差。

3. **辅助设备的自损耗计入规则**：
   储能柜内部的空调、风扇、BMS 等辅助设备的耗电在储能软件中可能被作为“内部损耗”直接抵扣（表现为放电量偏低、充电量偏高），而脚本处理的数据点只看系统总输出输入功率。

**总结**：这属于合理的数据采样精度误差。目前基于 5 分钟瞬时功率的脚本计算已达到该数据精度下的最优解；若要完全对齐，需要 CSV 数据源直接提供如“累计充电电量”、“累计放电电量”的连续累计字段。
