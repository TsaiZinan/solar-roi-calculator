# 每日收益分析 (Daily Analysis)

该技能用于自动执行每日的收益数据分析、报表更新及代码库同步。

## 触发条件
当用户要求“执行每日分析”、“更新今天的收益”、“生成每日报表”或提及“daily-analysis”时调用。

## 执行步骤

1. **运行数据分析脚本**：
   执行 `Script/daily_analysis.py` 或相关脚本，生成最新的每日收益分析报告。
   
2. **自动同步总报表**：
   根据最新生成的每日数据，将**核心利润**和**增益数据**自动同步更新到 `报告/总收益分析报表.md` 中。
   **注意**：更新时表头必须严格使用：
   - `总收益(光伏电价X元/度)`
   - `储能额外收益(光伏电价X元/度)`

3. **自动推送到 GitHub**：
   完成所有分析和报表更新后，必须自动执行以下 Git 命令将结果上传至远程仓库：
   ```bash
   git add 报告/ 数据/
   git commit -m "chore: update daily analysis report and data"
   git push origin main
   ```
   *确保将生成的结果和报表可靠地上传到 GitHub 仓库。*
